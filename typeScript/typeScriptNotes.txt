typeScript file compile to javaScript when we run it

steps:
# 1. "npm install -g typescript"  -> Install TypeScript globally
# 2. "tsc -v"                     -> Check if TypeScript is installed (should return a version)
# 3. "tsc yourfile.ts "           -> Compile your TypeScript file into JavaScript (generates yourfile.js in the same folder)
# 4. "node yourfile.js"           -> Run the compiled JavaScript file with Node.js
# 5. "npm install -g ts-node"     -> lets you run .ts files directly without compiling [(Optional) Install ts-node globally]
# 6. "ts-node yourfile.ts"        -> Run your TypeScript file directly


# Adding script inside package.json
{
  "scripts": {
    "start": "ts-node src/index.ts"
  }
}
# Now you can just run: "npm run start"

# 7. "npm link" -> Make your project a global command (# links this project globally so you can call it like a CLI tool)

Example:
If your project is called "name": "my-app" in package.json,
after npm link you can run:

my-app

✅ This way: Use tsc → compile TypeScript → run with Node.js.
